A plugin who displays a TV program for certain channels. This widget brings the TV Programs to you and your visitors.

== Description ==
This widget offers the possibility of embedding the Widget TV Program available on Port.ro into any WordPress site. Widget Program TV Port.ro is a free plugin who displays the TV program for 135 romanian and international channels. You can configure the display of the widget and customize : colors,fonts,size,width,height, border and choose what TV channels to display and how in order to fit best with any site design and structure.

Also you may choose from a list of 12 predefined templates : Now on TV, This evening on TV, Sport on TV, Documentary on TV, Cartoons on TV, Movies on TV,  etc. If you want a custom widget template with more channels, you may ask one at widget@port.ro or from widget.port.ro contact form available in user account interface.

This widget brings you the TV program for you and your visitors and it is a source of traffic for your website. Please rate our plugin if you find it useful and easy to configure. For more info about our TV listing and EPG offer click <a href= "http://media.port-network.com/port_ro/offers/epgdata.htm"  target="_blank"> here </a> .

== Installation ==

In order to use this plugin you have to create a new account at http://widget.port.ro and configure some TV programs.
Each configured Tv program provides you a unique id/code who can be used to setup a "Widget Program TV Port.ro" plugin.

1. Upload `WidgetPortRo.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag&drop the plugin where do you want and provide the widget id. The widget id can be retrieved from http://widget.port.ro after you login and create a widget.


Pentru instalarea şi configurarea plugin-ului de Wordpress Widget TV Port.ro trebuie să urmaţi paşii următori:
<br>
1. Configurati widget-ul dorit în interfața Widget TV Port.ro. Dacă nu aveți cont deja, puteți crea unul <a href="http://widget.port.ro"> AICI </a> . 
<br>
2. După ce widget-ul a fost configurat cu succes, notați ID-ul acestuia. 
<br>
3. Descărcați plugin-ul pentru Wordpress de <a href="http://wordpress.org/plugins/widget-program-tv-portro/" target="_blank" > AICI </a> sau de <a href="http://widget.port.ro/wpplugin" target="_blank" > AICI </a> și instalați-l. 
<br>
4. Activați Widget-ul TV Port.ro. 
<br>
5. Trageți plugin-ul Widget TV Port.ro și plasați-l în zona din sidebar unde doriți să apară. 
<br>
6. În dialogul de configurare al plugin-lui se completează cu ID-ul widget-ului, vezi punctul 2. 
<br>
Dacă totul s-a efectuat corect, în zona aleasă pentru instalarea plugin-ului va apărea conţinutul widget-ului PORT.ro 
<br>
